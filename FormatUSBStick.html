You can format a drive easily from the terminal:
sudo mkfs -t vfat -I /dev/sd##
This is format it to FAT 32.
For more details, Refer here.
to get the sd?? thing for your drive you can simply do:
df -h
About changing the labels(That is the name of the USB drive):
Ubuntu Documentation
I will not explain the whole process(its not that big) here. It is quite clearly given in the documentation.
Basically these are the steps:
1. Install the labeling program
2. Unmount the device
3. check the current label
4. change the label
5. mount it again to check if the present label has been effected.
Each step is clearly explained in the documentation. SO i think it would be unnecessary to go through all of it again.
for FAT partitions, the list of commands is:
sudo mlabel -i /device -s :: #to check the label
sudo mlabel -i /device ::NewLabel #to add the new label
